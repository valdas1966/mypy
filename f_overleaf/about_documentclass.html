\documentclass

The very first command in every LaTeX document

What Is It?

\documentclass is a mandatory command that must appear as the very first line of any LaTeX file. It tells LaTeX what type of document you're creating.

It controls: page size, font size defaults, heading styles, margins, available commands, and overall layout structure.

Syntax

Command \documentclass
Options (optional) [12pt, a4paper]
Class name {article}
\documentclass{article} % simplest form
\documentclass[12pt]{article} % with font size option
\documentclass[12pt,a4paper]{report} % multiple options

Common Options

10pt, 11pt, 12pt Base font size (default: 10pt)
a4paper, letterpaper Paper size
twocolumn Two-column layout
landscape Landscape orientation
draft Draft mode (shows overfull boxes, skips images)

Analogy

Think of it like choosing a template in Word or Google Docs

When you open Word and select "Blank Document" vs "Report" vs "Resume" — that's what \documentclass does. It picks the overall template that defines how your document looks and behaves before you write a single word.

Where It Fits

\documentclass{article} % ← FIRST LINE

% === PREAMBLE ===
% packages, settings, macros go here
\usepackage{graphicx}

% === CONTENT ===
\begin{document}
  Your content here...
\end{document}
