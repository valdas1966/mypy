Color Package

f_color

A. Purpose

  1. Color representation and visualization for the MyPy framework.
  2. Core class: RGB — float-based (0–1) with comparison, hashing, format conversion.
  3. Three-tier pattern: From (parse) / Factory (presets) / To (serialize).
  4. Utility: UColor — matplotlib-based color display.

B. Package Structure

C. Public API Overview

RGB (f_color.rgb)
  1. Construction: RGB(name='RED') or RGB(r=1, g=0, b=0).
  2. Properties: r, g, b, key.
  3. To: rgb.to.tuple(), rgb.to.hex(), rgb.to.argb().
  4. From: RGB.From.hex(), RGB.From.ints().
  5. Factory: gradient(), gradient_multi(), random().
  6. Mixins: Comparable + HasName — supports ==, <, hash().
UColor (f_color.u_color)
  1. UColor.show(rgbs) — displays colors as labeled matplotlib rectangles.

E. Usage Example

Quick start 
from f_color import RGB

# Create
red = RGB(name='RED')

# From external format
rgb = RGB.From.hex('#FF0000')

# Convert
red.to.hex()              # '#FF0000'
red.to.tuple(to_int=True) # (255, 0, 0)

# Gradient
gradient = RGB.Factory.gradient(
    a=RGB(name='BLACK'),
    b=RGB(name='WHITE'),
    n=5
)
