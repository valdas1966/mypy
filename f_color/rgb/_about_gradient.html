Gradient

Linear interpolation between two colors, step by step.

The Idea

A gradient takes color A and color B, and produces n evenly spaced colors between them. Each channel (R, G, B) is interpolated independently using numpy.linspace.

gradient(a=RGB(name='BLACK'), b=RGB(name='WHITE'), n=5)

# Produces 5 colors:
# (0.0, 0.0, 0.0)  →  (0.25, 0.25, 0.25)  →  (0.5, 0.5, 0.5)  →  (0.75, 0.75, 0.75)  →  (1.0, 1.0, 1.0)

How It Works

Each channel is a straight line from A's value to B's value, sampled at n equally spaced points.

Math For channel value v at step i of n:  v = a + (b - a) × i / (n - 1)

Edge Cases

# n=1 → returns only color A
gradient(a=red, b=blue, n=1)   # [red]

# n=2 → returns exactly [A, B]
gradient(a=red, b=blue, n=2)   # [red, blue]

# A == B → all n colors are identical
gradient(a=red, b=red, n=5)    # [red, red, red, red, red]

gradient_multi — Multiple Stops

Chains multiple gradient() calls across a list of color stops. Each segment gets an equal share of n, and boundary colors are shared (not duplicated).

Usage

from f_color.rgb import RGB

# Two-color gradient
black = RGB(name='BLACK')
white = RGB(name='WHITE')
colors = RGB.Factory.gradient(a=black, b=white, n=10)

# Multi-stop gradient (e.g. heatmap: green → yellow → red)
stops = [RGB(name='GREEN'), RGB(name='YELLOW'), RGB(name='RED')]
heatmap = RGB.Factory.gradient_multi(stops=stops, n=20)

# Access individual colors
for c in colors:
    print(c.to.hex())        # '#000000', '#1C1C1C', ...
    print(c.to.tuple())      # (0.0, 0.0, 0.0), (0.11, 0.11, 0.11), ...

Facts & Trivia

  • Linear vs. Perceptual RGB gradients interpolate in RGB space, which is not perceptually uniform. A gradient from green to red passes through muddy brown, not vibrant yellow. Perceptual color spaces like OKLCH solve this — but RGB is simpler and fast.
  • Why linspace? numpy.linspace guarantees the first value is exactly A and the last is exactly B, with perfect spacing. No off-by-one, no floating-point drift at the edges.
  • Heatmaps Most heatmaps are just gradient_multi under the hood: blue → cyan → green → yellow → red. Matplotlib's "jet" colormap works the same way (and has been controversial since the 90s for being perceptually misleading).
  • Film The famous "color ramp" effect in movies (teal shadows + orange highlights) is literally a two-stop gradient applied as a color grade. Hollywood discovered that teal-orange spans the widest perceptual range on human skin tones.
  • Fun Photoshop's gradient tool was added in version 2.5 (1993). Before that, designers faked gradients by hand-painting bands of color — or used actual airbrushes on physical prints.
