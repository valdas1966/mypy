RGB Color

f_color / rgb

A. Purpose

  1. RGB color representation with float values (0–1).
  2. Supports construction from name (matplotlib/custom), hex string, or raw floats.
  3. Three-tier conversion: From (parse), Factory (presets), To (serialize).

E. Usage Examples

Construction and conversion 
from f_color.rgb import RGB

red = RGB(name='RED')
gray = RGB(r=0.5, g=0.5, b=0.5)

# From external formats
rgb = RGB.From.hex('#FF0000')
rgb = RGB.From.ints(r=255, g=0, b=0)

# To external formats
red.to.hex()                # '#FF0000'
red.to.tuple(to_int=True)   # (255, 0, 0)
red.to.argb()               # 'FFFF0000'
Gradient generation 
from f_color.rgb import RGB

black = RGB.Factory.black()
white = RGB.Factory.white()
gradient = RGB.Factory.gradient(a=black, b=white, n=5)
# [RGB(0,0,0), RGB(.25,.25,.25), ..., RGB(1,1,1)]
