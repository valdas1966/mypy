RGB Color

f_color / rgb

A. Purpose

  1. RGB color representation with float channels (0–1), comparable and hashable via (r, g, b) key.
  2. Construct from matplotlib color name, custom palette name, hex string, int values, or raw floats.
  3. Three-tier pattern: From (parse in), Factory (generate), To (serialize out).

E. Usage Examples

Construction & Conversion 
from f_color.rgb import RGB

# From name
red = RGB(name='RED')
str(red)                    # 'RED(255, 0, 0)'
red.to.hex()                # '#FF0000'
red.to.tuple(to_int=True)   # (255, 0, 0)
red.to.argb()               # 'FFFF0000'

# From raw floats
gray = RGB(r=0.5, g=0.5, b=0.5)
str(gray)                   # '(127, 127, 127)'

# From external formats
rgb = RGB.From.hex('#FF0000')
rgb = RGB.From.ints(r=255, g=0, b=0)
Comparison & Gradient 
from f_color.rgb import RGB

# Comparison
assert RGB(name='BLACK') < RGB(name='WHITE')
assert RGB(name='BLACK') == RGB(r=0, g=0, b=0)

# Gradient: 5 steps from white to black
gradient = RGB.Factory.gradient(
    a=RGB(name='WHITE'),
    b=RGB(name='BLACK'),
    n=5
)
assert gradient[0] == RGB(r=1, g=1, b=1)
assert gradient[2] == RGB(r=0.5, g=0.5, b=0.5)
assert gradient[4] == RGB(r=0, g=0, b=0)
