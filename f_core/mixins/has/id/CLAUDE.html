HasId

f_core/mixins/has/id/
MIXIN

Purpose

HasId provides lightweight identity semantics for any object that needs a unique birth timestamp and optional group/counter identifiers. On every instantiation it captures a nanosecond-precision datetime string, giving each instance a naturally-ordered, collision-resistant birth ID without requiring external UUID libraries.

The optional id_group and id_counter parameters allow callers to attach categorical and sequential identity dimensions alongside the temporal one.

HasId is a plain class (not a mixin that inherits from another base). It is designed to be used as a base class or mixed into inheritance chains that need identity tracking.

Constructor & Properties ▼

Constructor

HasId(id_group: str = None, id_counter: int = None) -> None

Creates a new HasId instance. Automatically captures the current datetime at nanosecond precision via UDateTime.str_now(UDateTime.Format.NANO) and stores it as the birth ID.

Parameter Type Default Description
id_group str None Optional categorical group identifier for the instance.
id_counter int None Optional sequential counter identifier for the instance.

PROPERTY id_birth

@property id_birth -> str

Returns the nanosecond-precision datetime string captured at the moment the instance was created. This value is set once during __init__ and is read-only.

PROPERTY id_group

@property id_group -> str

Returns the group identifier string passed at construction time, or None if no group was specified.

PROPERTY id_counter

@property id_counter -> int

Returns the integer counter passed at construction time, or None if no counter was specified.

Usage Examples

Example 1: Basic Instantiation ▼

Create a bare HasId instance with no group or counter. The birth timestamp is generated automatically. 

from f_core.mixins.has.id import HasId

obj = HasId()
print(obj.id_birth)      # e.g. "2026-02-16 14:30:05.123456789"
print(obj.id_group)      # None
print(obj.id_counter)    # None
Example 2: With Group and Counter ▼

Supply an id_group and id_counter to attach categorical and sequential dimensions to the identity. 

from f_core.mixins.has.id import HasId

obj = HasId(id_group="batch_alpha", id_counter=42)
print(obj.id_birth)      # nanosecond timestamp
print(obj.id_group)      # "batch_alpha"
print(obj.id_counter)    # 42
