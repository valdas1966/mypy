A. Purpose

  1. HasRowCol is a mixin that gives objects row and col properties for 2D grid positioning.
  2. Provides row-major comparison, hashing, clockwise neighbor discovery, Manhattan distance, and bounds checking.
  3. All operators delegate to the (row, col) key tuple.

E. Usage Examples

Basic Usage 
from f_core.mixins.has.row_col import HasRowCol

cell = HasRowCol(row=1, col=2)
print(cell)           # (1,2)
print(cell.key)       # (1, 2)
print(repr(cell))     # <HasRowCol: Row=1, Col=2>
Comparison and Hashing 
a = HasRowCol(0, 0)
b = HasRowCol(1, 1)

a < b                        # True (row-major)
a == HasRowCol(0, 0)         # True
hash(a) != hash(b)           # True
{a, b, HasRowCol(0, 0)}     # {(0,0), (1,1)} — dedup works
Neighbors and Distance 
cell = HasRowCol(1, 2)
cell.neighbors()
# [(0,2), (1,3), (2,2), (1,1)]  — N, E, S, W

cell.distance(HasRowCol(3, 4))  # 4
Bounds Checking 
from f_core.mixins.has.row_col import HasRowCol
from f_math.shapes import Rect

cell = HasRowCol(1, 2)
rect = Rect.From.Center(x=0, y=0, distance=2)
cell.is_within(rect=rect)  # True

cell.is_within(row_min=0, col_min=0,
               row_max=5, col_max=5)  # True
