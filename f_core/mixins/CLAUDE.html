A. Purpose

  1. Library of 13 composable mixin classes that inject standard behaviors via multiple inheritance.
  2. Three categories: equality/ordering (Equatable, Comparable, Hashable), size/validity (Sizable, Dictable, Validatable), property injection (Has* mixins).
  3. Designed for composition — mix any combination into a single domain class.

Equality / Ordering Mixins 
# Equatable — abstract key, concrete __eq__
@property
@abstractmethod
def key(self) -> SupportsEquality

def __eq__(self, other: object) -> bool
  1. Identity short-circuit: returns True if other is self.
# Comparable — extends Equatable with ordering
def __lt__(self, other: object) -> bool
def __le__(self, other: object) -> bool
def __gt__(self, other: object) -> bool
def __ge__(self, other: object) -> bool
  1. All operators delegate to self.key vs other.key.
# Hashable — extends Equatable with hashing
def __hash__(self) -> int
  1. Returns hash(self.key). Guarantees a == b implies hash(a) == hash(b).

Decision Guide
  1. Need equality only? Extend Equatable, implement key.
  2. Need sorting/ordering? Extend Comparable.
  3. Need sets/dicts? Extend Hashable (or Comparable + Hashable).
  4. Need len/bool? Extend Sizable, implement __len__.
  5. Need dict-like access? Extend Dictable[K, V].
  6. Need validity flag? Use Validatable (immutable) or ValidatableMutable (mutable).
  7. Need identity from a value? Use HasKey[Key].
  8. Need named objects? Use HasName.
  9. Need 2D position? Use HasRowCol.
  10. Need 2D dimensions? Use HasRowsCols.

G. Usage Examples

Equality and Comparison 
from f_core.mixins import Comparable

class Score(Comparable):
    def __init__(self, points: int) -> None:
        self.points = points

    @property
    def key(self) -> int:
        return self.points

s1, s2 = Score(100), Score(150)
s1 < s2              # True
s1 == Score(100)      # True
sorted([Score(50), Score(100), Score(75)])  # [50, 75, 100]
Dict-like Object 
from f_core.mixins import Dictable

d = Dictable(data={'a': 1, 'b': 2})
d['c'] = 3
print(len(d))          # 3
print(bool(d))         # True
print('a' in d)        # True
print(d.get('z', 0))   # 0
Mutable Validity 
from f_core.mixins import ValidatableMutable

obj = ValidatableMutable(is_valid=True)
if obj:                # True
    obj.set_invalid()
if not obj:            # True
    obj.set_valid()
Mixin Composition 
from f_core.mixins import HasRowCol, ValidatableMutable
from f_core.mixins.has.name import HasName

class Cell(HasRowCol, HasName, ValidatableMutable):
    def __init__(self, row: int, col: int,
                 name: str, is_valid: bool = True) -> None:
        HasRowCol.__init__(self, row=row, col=col)
        HasName.__init__(self, name=name)
        ValidatableMutable.__init__(self, is_valid=is_valid)

cell = Cell(row=1, col=2, name='Start')
print(cell.name)       # 'Start'
print(cell.key)        # (1, 2)
print(bool(cell))      # True
cell.set_invalid()
print(bool(cell))      # False
Named Identity 
from f_core.mixins.has.key import HasKey

a = HasKey[str](key='A')
b = HasKey[str](key='B')

print(a < b)          # True
print(hash(a))         # hash('A')
print({a, b, HasKey[str](key='A')})  # {A, B}
