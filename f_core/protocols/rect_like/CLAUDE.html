A. Purpose

  1. Protocol for objects convertible to a min/max rectangle representation.
  2. Used by HasRowCol.is_within() for bounds checking against rectangular regions.

E. Usage Examples

Type hint for bounds checking 
from f_core.protocols.rect_like import RectLike


def is_within(row: int, col: int, rect: RectLike) -> bool:
    x_min, y_min, x_max, y_max = rect.to_rect_coords()
    return x_min <= row <= x_max and y_min <= col <= y_max
Used by HasRowCol
