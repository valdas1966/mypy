Bounds & SupportsBounds

f_core/protocols/bounds/
NAMEDTUPLE PROTOCOL

Purpose

Defines a Bounds NamedTuple for rectangular bounding boxes (top, left, bottom, right) and a SupportsBounds Protocol for objects that can provide their bounds. Generic over int or float.

Bounds is the data container; SupportsBounds is the structural protocol. Any class with a bounds() → Bounds[T] method satisfies the protocol without explicit inheritance.

Usage Examples

Example 1: Create and use Bounds directly ▼ 
from f_core.protocols.bounds.main import Bounds

# Create bounds
b = Bounds(top=0, left=0, bottom=100, right=200)

# Access by name
print(b.top, b.right)       # 0 200

# Unpack like a tuple
top, left, bottom, right = b

# Compute area
area = (b.bottom - b.top) * (b.right - b.left)  # 20000
Example 2: Use SupportsBounds as type hint ▼ 
from f_core.protocols.bounds.main import Bounds, SupportsBounds


def is_within(row: int, col: int, obj: SupportsBounds[int]) -> bool:
    b = obj.bounds()
    return b.top <= row <= b.bottom and b.left <= col <= b.right


# Any object with a bounds() -> Bounds[int] method satisfies the protocol
