A. Purpose

  1. ProcessBase is the abstract base class for all process classes in f_core.
  2. Uses the Template Method pattern: run() orchestrates _run_pre() → _run() → _run_post().
  3. Provides execution timing (elapsed) and a lap timer (seconds_since_last_call()).

B. Public API

Constructor 
def __init__(self, name: str = 'ProcessBase') -> None
  1. Calls HasName.__init__ and ValidatableMutable.__init__.
  2. Initializes _elapsed, _time_start, _time_finish to None.
  3. Initializes lap timer checkpoint _time_lap_prev to None.
run() 
def run(self) -> None
  1. Entry point for process execution.
  2. Calls _run_pre(), _run(), _run_post() in sequence.
  3. Rarely overridden; override _run() instead.
elapsed (property) 
@property
def elapsed(self) -> int
  1. Returns total execution time in seconds (truncated to int).
  2. None before run() completes.
seconds_since_last_call() 
def seconds_since_last_call(self) -> int
  1. Returns seconds elapsed since the previous call to this method.
  2. First call returns 0 and stores a checkpoint.
  3. Subsequent calls return delta and update the checkpoint.

E. Usage Examples

Basic Subclass 
from f_core.processes.i_0_base import ProcessBase

class MyProcess(ProcessBase):
    def _run(self) -> None:
        # process logic here
        pass

process = MyProcess(name='DataProcessor')
process.run()
print(process.elapsed)  # seconds taken
Lap Timer 
import time
from f_core.processes.i_0_base import ProcessBase

class TimedSteps(ProcessBase):
    def _run(self) -> None:
        self.seconds_since_last_call()  # 0 (first)
        time.sleep(2)
        print(self.seconds_since_last_call())  # 2
        time.sleep(3)
        print(self.seconds_since_last_call())  # 3

TimedSteps(name='Steps').run()
