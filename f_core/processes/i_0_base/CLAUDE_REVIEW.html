9. Usage & Usefulness

  1. How to use: inherit from ProcessBase, override _run(). Call run() to execute with timing.
  2. When useful: any operation that needs lifecycle hooks and/or execution timing. Root of the process hierarchy used by ProcessInput, ProcessOutput, ProcessIO.
  3. Overengineered? No. Minimal base class with timing. Lap timer is a small addition that costs nothing when unused.
  4. Alternative: a plain function with time.time() calls. The class hierarchy is justified by the downstream generic processes that build on it.
