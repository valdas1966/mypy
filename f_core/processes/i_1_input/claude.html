A. Purpose

  1. ProcessInput[Input] is a generic process that accepts typed input data.
  2. Extends ProcessBase with a read-only input property.
  3. Level 1 process — single capability: input handling.

E. Usage Examples

Basic Subclass 
from f_core.processes.i_1_input import ProcessInput

class DataLoader(ProcessInput[str]):
    def _run(self) -> None:
        filepath = self.input
        print(f'Loading: {filepath}')

loader = DataLoader(input='/data/file.csv', name='Loader')
loader.run()
print(loader.elapsed)  # seconds taken
With Integer Input 
class Doubler(ProcessInput[int]):
    def _run(self) -> None:
        print(self.input * 2)

Doubler(input=5, name='Doubler').run()  # prints 10
Factory 
from f_core.processes.i_1_input import ProcessInput

Proc = ProcessInput.Factory.a()
