A. Purpose

  1. Process framework providing reusable base classes with Template Method lifecycle.
  2. All processes share: run() entry point, automatic elapsed timing, name-based identity.
  3. 5 classes in a clean inheritance hierarchy, from abstract base to parallel execution.

E. Lifecycle

Template Method Pattern
  1. run() — entry point, orchestrates the lifecycle.
  2. _run_pre() — resets elapsed, records start time.
  3. _run() — override this with process logic. Return output for IO/Parallel.
  4. _run_post() — records finish time, calculates elapsed.
def run(self) -> None | Output:
    self._run_pre()        # reset timing
    result = self._run()   # YOUR LOGIC HERE
    self._run_post()       # calculate elapsed
    return result          # (ProcessOutput and below)

F. Class Comparison

Feature Matrix
ClassType ParamsInputOutputParallelUse Case
ProcessBase —NoNoNo Logging, cleanup, side effects
ProcessInput [Input]YesNoNo Consume data, no return value
ProcessOutput [Output]NoYesNo Generate data from nothing
ProcessIO [In, Out]YesYesNo Transform input to output
ProcessParallel [Item, Out]YesYesYes Distribute work across workers
Decision guide
  1. Need input + output? Use ProcessIO (most common).
  2. Need parallelism? Use ProcessParallel with a func.
  3. Only need input? Use ProcessInput (e.g., data loader).
  4. Only need output? Use ProcessOutput (e.g., generator).
  5. Just side effects? Use ProcessBase directly.

H. Usage Examples

Transform (ProcessIO) 
from f_core.processes import ProcessIO

class Doubler(ProcessIO[int, int]):
    def _run(self) -> int:
        return self.input * 2

result = Doubler(input=5, name='Doubler').run()
print(result)         # 10
print(proc.elapsed)   # 0 (seconds)
Parallel Execution (ProcessParallel) 
from f_core.processes import ProcessParallel

def square_chunk(chunk: list[int]) -> int:
    return sum(x * x for x in chunk)

proc = ProcessParallel(input=list(range(1, 13)),
                       func=square_chunk,
                       workers=3)
output = proc.run()
print(output)        # [14, 77, 194]
print(proc.errors)   # []
Side-Effect Process (ProcessBase) 
from f_core.processes.i_0_base import ProcessBase

class Logger(ProcessBase):
    def _run(self) -> None:
        print(f'{self.name} executed')

Logger(name='MyLogger').run()
print(Logger.elapsed)  # 0
Generator (ProcessOutput) 
from f_core.processes.i_1_output import ProcessOutput

class RangeGen(ProcessOutput[list[int]]):
    def _run(self) -> list[int]:
        return list(range(10))

result = RangeGen(name='Gen').run()
print(result)  # [0, 1, 2, ..., 9]
