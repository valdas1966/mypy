What is ProcessABC?

The abstract base class for all processes in f_core. Provides timing, lifecycle management, and inherits name-based comparison from HasName and validity checking from Validatable.

Timing

elapsed property tracks execution time

Lifecycle

Template method: pre → run → post

Lap Timer

seconds_since_last_call()

Sortable

Compare processes by name

Template Method Pattern

1

_run_pre()

Setup: reset timing, record start time

↓
2

_run() Override This

Main process logic - implement in subclass

↓
3

_run_post()

Cleanup: record finish, calculate elapsed

Quick Start

from f_core.processes.i_0_abc import ProcessABC

class MyProcess(ProcessABC):
    def _run(self) -> None:
        # Your process logic here
        self.do_work()

process = MyProcess(name='DataProcessor')
process.run()
print(f"Completed in {process.elapsed} seconds")

Execution Flow

run()
→
_run_pre()
→
_run()
→
_run_post()
Method Actions Override?
run() Orchestrates lifecycle Rarely
_run_pre() _elapsed = None, _time_start = time() Rarely
_run() Empty - your logic goes here Yes
_run_post() _time_finish = time(), calculate elapsed Rarely

Timing Attributes

Attribute Type Set By Description
_time_start float | None _run_pre() Unix timestamp at start
_time_finish float | None _run_post() Unix timestamp at finish
_elapsed int | None _run_post() Total seconds (truncated)
elapsed int | None Property Read-only access to _elapsed

Lap Timer

seconds_since_last_call() measures time between consecutive calls.

Call 1
→ 0
Call 2
→ 2 sec
Call 3
→ 3 sec
Call 4
→ 1 sec 
def _run(self):
    print(self.seconds_since_last_call())  # 0 (first call)
    time.sleep(2)
    print(self.seconds_since_last_call())  # 2
    time.sleep(3)
    print(self.seconds_since_last_call())  # 3

Inheritance Chain

Equable ─ ==, !=, hash() └── Comparable ─ <, <=, >, >= └── HasName ─ name, str(), repr() │ └── ProcessABC ─ run(), elapsed, timing │ └── Validatable ─ __bool__(), _is_valid

Process Hierarchy

ProcessABC ─ i_0_abc (you are here) ├── ProcessInput[Input] ─ i_1_input │ └── Adds: input property │ ├── ProcessOutput[Output] ─ i_1_output │ └── Adds: output, run() returns Output │ └── ProcessIO[Input, Output] ─ i_2_io └── Combines: input + output

Naming Convention

Prefix Level Description
i_0_ Base Abstract base class

i_2_ Level 2 Combined capabilities

Basic Subclass

from f_core.processes.i_0_abc import ProcessABC

class DataProcessor(ProcessABC):
    def _run(self) -> None:
        # Your processing logic
        self.load_data()
        self.transform_data()
        self.save_data()

processor = DataProcessor(name='ETL Pipeline')
processor.run()

print(f"Process: {processor.name}")
print(f"Elapsed: {processor.elapsed} seconds")

With Lap Timer

class TimedProcess(ProcessABC):
    def _run(self) -> None:
        # Track time for each step
        self.seconds_since_last_call()  # Reset (returns 0)

        self.step_one()
        print(f"Step 1: {self.seconds_since_last_call()}s")

        self.step_two()
        print(f"Step 2: {self.seconds_since_last_call()}s")

        self.step_three()
        print(f"Step 3: {self.seconds_since_last_call()}s")

Sorting Processes

# Processes can be sorted by name (from HasName)
processes = [
    DataProcessor(name='Zebra'),
    DataProcessor(name='Alpha'),
    DataProcessor(name='Beta')
]

sorted(processes)  # [Alpha, Beta, Zebra]
min(processes)     # Alpha
max(processes)     # Zebra

# In sets (hashable)
unique = {processes[0], processes[1]}

Validity Check

# From Validatable
process = DataProcessor(name='Test')

# Check validity before running
if process:  # Calls __bool__() → _is_valid
    process.run()
else:
    print("Process not valid")
