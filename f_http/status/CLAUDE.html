Status

f_http/status/

1. Purpose

  1. Wraps Python's http.HTTPStatus with validation and a readable name.
  2. A Status is truthy only when the code is in the 2xx range (200-299).
  3. Unknown codes resolve to name 'UNKNOWN' and are invalid.

5. Usage Examples

Example 1 — Valid Status (200 OK) 
from f_http.status import Status

status = Status(code=200)
assert status                          # True (2xx)
assert status.code == 200
assert status.name == 'OK'
assert str(status) == 'Code=200, Name=OK'
assert repr(status) == '<Status: Name=OK>'
Example 2 — Invalid Status (404 Not Found) 
from f_http.status import Status

status = Status(code=404)
assert not status                      # False (not 2xx)
assert status.name == 'NOT_FOUND'
assert str(status) == 'Code=404, Name=NOT_FOUND'
Example 3 — Unknown Status (None) 
from f_http.status import Status

status = Status(code=None)
assert not status
assert status.name == 'UNKNOWN'
assert str(status) == 'Code=None, Name=UNKNOWN'
