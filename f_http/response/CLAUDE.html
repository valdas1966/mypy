Response

f_http/response/

1. Purpose

  1. Immutable result object returned by Client.get_json().
  2. Wraps HTTP status, parsed JSON data, elapsed time, and exception message.
  3. Valid (truthy) when status is 2xx and data is not None.

5. Usage Examples

OK Response (valid) 
from f_http.response import Response

response = Response.Factory.ok()
assert response                         # True (2xx + data)
assert response.status.code == 200
assert response.status.name == 'OK'
assert response.data == {'key': 'value'}
assert response.elapsed == 0.1
assert response.exception is None
  1. Factory creates a 200 OK response with sample data.
  2. Truthiness confirms both status and data are present.
Not Found Response (invalid) 
from f_http.response import Response

response = Response.Factory.not_found()
assert not response                     # False (404 + no data)
assert response.status.code == 404
assert response.status.name == 'NOT_FOUND'
assert response.data is None
  1. 404 status makes the response invalid (not 2xx).
  2. Data is None, reinforcing invalidity.
Unknown Error Response (invalid) 
from f_http.response import Response

response = Response.Factory.unknown()
assert not response                     # False (no status)
assert response.status.code is None
assert response.exception == 'Unknown error'
  1. No HTTP status code; exception carries the error message.
