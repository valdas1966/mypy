Bounds

f_ds/geometry/bounds/
DATA CONTAINER GENERIC[T] IMMUTABLE

Purpose

Immutable generic container for rectangular bounds defined by four coordinates (top, left, bottom, right). Generic over int or float. Validates that top <= bottom and left <= right at construction time. Used as the fundamental coordinate primitive for 2D positioning throughout the framework.

All four coordinates are read-only after construction. No setters are provided. Constraints are enforced via assertions in __init__.

Usage Examples

Example 1: Create and access bounds ▼ 
from f_ds.geometry.bounds import Bounds

# Create bounds directly
b = Bounds(top=10, left=20, bottom=50, right=80)

# Access by property
print(b.top)         # 10
print(b.right)       # 80

# Convert to tuple
print(b.to_tuple())  # (10, 20, 50, 80)

# String representations
print(str(b))        # (10, 20, 50, 80)
print(repr(b))       # <Bounds: Top=10, Left=20, Bottom=50, Right=80>
Example 2: Using the Factory ▼ 
from f_ds.geometry.bounds import Bounds

# Full bounds (0-100 range)
full = Bounds.Factory.full()
print(full.to_tuple())  # (0, 0, 100, 100)
print(str(full))        # (0, 0, 100, 100)
