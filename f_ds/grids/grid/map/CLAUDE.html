GridMap

f_ds/grids/grid/map/
2D GRID GENERIC[CellMap] MAP / OBSTACLES

Purpose

  1. 2D grid of CellMap cells where each cell can be valid or invalid (obstacle).
  2. Extends GridBase[CellMap] with map-specific operations: valid-cell filtering, validity-aware neighbors, random sampling, obstacle invalidation, and analytic reporting.
  3. Companion classes: Factory (preset grids), From (construct from files/arrays), Random (cell sampling).

Usage Examples

Example 1: Factory presets and basic queries ▼ 
from f_ds.grids.grid.map import GridMap

# Create a 3x3 X-shaped grid (5 valid cells)
grid = GridMap.Factory.x()
print(str(grid))                   # GridMap(3x3, 5)
print(repr(grid))                  # <GridMap: Name=GridMap, Shape=3x3, Cells=5>
print(len(grid))                   # 5
print(len(grid.cells_valid()))     # 5
print(len(grid.neighbors(grid[1][1])))  # valid neighbors of center
Example 2: Analytics reporting ▼ 
from f_ds.grids.grid.map import GridMap

grid = GridMap(rows=4, cols=4, name='Arena', domain='game')
analytics = grid.to_analytics()
# {'domain': 'game', 'map': 'Arena', 'rows': 4, 'cols': 4, 'cells': 16}
