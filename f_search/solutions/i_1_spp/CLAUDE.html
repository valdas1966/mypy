SolutionSPP

A. Purpose

  1. Solution container for One-to-One Shortest Path Problem results.
  2. Extends SolutionSearch with path, optimal goal cost, and heuristic quality metric.
  3. Immutable after creation; represents a final algorithm result.

SolutionSPP

             name_algo: str,
             problem: Problem,
             is_valid: bool,
             path: Path = None,
             g_goal: int = None,
             stats: StatsSearch = None) -> None
  1. Delegates name_algo, problem, is_valid, stats to SolutionSearch.
  2. Defaults stats to a fresh StatsSearch() if not provided.
  3. Stores path and g_goal as private attributes.

path

@property
def path(self) -> Path
  1. Returns the solution path from start to goal.

g_goal

@property
def g_goal(self) -> int | None
  1. Returns the optimal cost to reach the goal.
  2. None if not set.

quality_h

@property
def quality_h(self) -> float | None
  1. Returns heuristic quality as h(start) / g(goal) in [0, 1].
  2. Returns None if solution is invalid or g_goal is 0 or None.
  3. Uses self.problem.h_start for the heuristic estimate.

E. Usage Examples

Factory Usage 
from f_search.solutions.i_1_spp import SolutionSPP

# Create via Factory
valid = SolutionSPP.Factory.valid()
invalid = SolutionSPP.Factory.invalid()

# Check result
if valid.is_valid:
    print(f"Path: {valid.path}")
    print(f"Goal cost: {valid.g_goal}")
    print(f"Heuristic quality: {valid.quality_h}")
    print(f"Explored: {valid.stats.explored}")
Direct Construction 
from f_search.solutions.i_1_spp.main import SolutionSPP
from f_search.ds.path import Path
from f_search.stats import StatsSearch

solution = SolutionSPP(
    name_algo='AStar',
    problem=problem,
    is_valid=True,
    path=path,
    g_goal=42,
    stats=stats
)

# Heuristic quality: h(start) / g(goal)
print(solution.quality_h)  # float in [0, 1]
