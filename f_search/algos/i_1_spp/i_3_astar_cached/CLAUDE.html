AStarCached

f_search/algos/i_1_spp/i_3_astar_cached/

1. Purpose

  1. A* variant that prioritizes states with cached (exact) heuristics over bounded and unbounded ones.
  2. Uses PriorityGHFlags for flag-aware frontier ordering and early termination.
  3. Extends AStarReusable; adds DataCached for cached/bounded heuristic storage.

5. Usage Examples

Without Cache (standard A* behavior) 
from f_search.algos.i_1_spp.i_3_astar_cached import AStarCached
from f_search.problems import ProblemSPP

problem = ProblemSPP.Factory.for_cached()
algo = AStarCached(problem=problem)
sol = algo.run()
# sol.stats.explored == 22
  1. No cached data provided; behaves like standard A*.
With Cached Exact Distances 
from f_search.algos.i_1_spp.i_3_astar_cached import AStarCached
from f_search.problems import ProblemSPP
from f_search.ds.data.cached import DataCached

problem = ProblemSPP.Factory.for_cached()
data_cached = DataCached.Factory.six_cached()
algo = AStarCached(problem=problem, data_cached=data_cached)
sol = algo.run()
# sol.stats.explored == 10  (fewer states explored)
  1. Cached heuristics enable early termination, reducing explored states from 22 to 10.
With Bounded Heuristics 
problem = ProblemSPP.Factory.for_cached()
data_cached = DataCached.Factory.six_bounded()
algo = AStarCached(problem=problem, data_cached=data_cached)
sol = algo.run()
# sol.stats.explored == 8
  1. Bounded heuristics further improve frontier ordering, reducing explored states to 8.
