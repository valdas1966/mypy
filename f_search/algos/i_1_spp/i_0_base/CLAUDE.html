f_search/algos/i_1_spp/i_0_base/

A. Purpose

  1. Abstract base for One-to-One Shortest-Path-Problem algorithms.
  2. Implements _run() loop: discover start, select-best, terminate-or-explore, return SolutionSPP.
  3. Extracts g_goal (optimal cost to goal) from dict_g on valid termination and passes it to the solution.

_run

def _run(self) -> SolutionSPP:
  1. Discovers start state if frontier is empty.
  2. Loops: selects best state, checks termination, explores best.
  3. On valid solution, computes g_goal = self._data.dict_g[self.problem.goal].
  4. Optionally reconstructs path via self._data.path_to(state=goal).
  5. Returns SolutionSPP(name_algo, problem, is_valid, path, g_goal, stats).

_can_terminate

def _can_terminate(self) -> bool:
  1. Returns True when self._data.best == self.problem.goal.

E. Usage Examples

Running an SPP Algorithm (A*)

from f_search.algos.i_1_spp.i_1_astar import AStar
from f_search.problems import ProblemSPP

problem = ProblemSPP.Factory.without_obstacles()
astar = AStar(problem=problem)
solution = astar.run()

if solution.is_valid:
    print(solution.g_goal)   # Optimal cost to reach goal
    print(solution.path)     # Path from start to goal
  1. AStar extends AlgoSPP; its run() calls _run().
  2. solution.g_goal contains the optimal path cost extracted from dict_g.
  3. solution.path is reconstructed only if _need_path is set.
