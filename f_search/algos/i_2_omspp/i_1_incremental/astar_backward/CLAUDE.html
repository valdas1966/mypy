AStarIncrementalBackward

f_search / algos / i_2_omspp / i_1_incremental / astar_backward
ALGORITHM OMSPP

Purpose

  1. Backward incremental A* for One-to-Many Shortest-Path-Problem (OMSPP).
  2. Converts OMSPP to Many-to-One by reversing each sub-problem; runs backward A* from each goal toward the shared start.
  3. Accumulates exact (cached) and lower-bound (bounded) heuristic info across sub-searches via DataCached.

PRIVATE _run

def _run(self) -> SolutionOMSPP:
  1. Converts OMSPP to list of forward SPPs.
  2. Sorts goals by ascending Manhattan distance to start.
  3. Runs backward A* for each goal via _run_sub_search.
  4. Accumulates sub-solutions and returns SolutionOMSPP.

PRIVATE _run_sub_search

def _run_sub_search(self, forward_problem: ProblemSPP, backward_problem: ProblemSPP, is_last: bool = False ) -> SolutionSPP | None:
  1. Runs backward AStarCached from goal Gi toward start S.
  2. If goal reached and not last: extracts cached (exact) distances for path states.
  3. If with_bounds=True: extracts bounded (lower-bound) distances for non-path states.
  4. Keeps the tighter (max) lower bound per state across sub-searches.
  5. Returns a forward SolutionSPP with optional reversed path.

Usage Examples

Example 1 — Basic Run (from _tester.py) ▼ 
from f_search.algos.i_2_omspp.i_1_incremental.astar_backward import (
    AStarIncrementalBackward)
from f_search.problems.i_2_omspp import ProblemOMSPP

problem = ProblemOMSPP.Factory.for_cached()

# Without bounds: 31 explored states
algo = AStarIncrementalBackward(problem=problem, with_bounds=False)
sol = algo.run()
assert sol.stats.explored == 31

# With bounds: 25 explored states (tighter heuristics)
algo = AStarIncrementalBackward(problem=problem, with_bounds=True)
sol = algo.run()
assert sol.stats.explored == 25
Example 2 — Path Reconstruction (from _tester.py) ▼ 
from f_search.algos.i_2_omspp.i_1_incremental.astar_backward import (
    AStarIncrementalBackward)
from f_search.problems.i_2_omspp import ProblemOMSPP

problem = ProblemOMSPP.Factory.custom(rows=10,
                                       pct_obstacles=20,
                                       k=5)
algo = AStarIncrementalBackward(problem=problem, need_path=True)
solution = algo.run()

if solution:
    for goal in problem.goals:
        path = solution.paths[goal]
        assert path.head() == problem.start
        assert path.tail() == goal

solution = algo.run()
