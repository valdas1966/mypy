ProblemNeighborhood

f_search/problems/i_1_neighborhood/

A. Purpose

  1. Defines a grid-based search problem that explores states within a bounded step count from a start.
  2. Composes ProblemSearch (grid + successors) with HasStart (start state) and adds steps_max.
  3. Constructor accepts only Grid (not Grid | str); inherits pickle support from ProblemSearch.

E. Usage Examples

Example 1 — Via Factory 
from f_search.problems.i_1_neighborhood import ProblemNeighborhood

problem = ProblemNeighborhood.Factory.without_obstacles()
print(problem.start)       # StateCell at grid[0][0]
print(problem.steps_max)   # 1
print(problem.grid)        # 4x4 grid without obstacles
Example 2 — Direct Construction 
from f_ds.grids import GridMap as Grid
from f_search.ds.state import StateCell as State
from f_search.problems.i_1_neighborhood import ProblemNeighborhood

grid = Grid.Factory.four_without_obstacles()
start = State(key=grid[0][0])
problem = ProblemNeighborhood(grid=grid, start=start, steps_max=3)

successors = problem.successors(start)  # neighbor states
