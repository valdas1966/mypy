ProblemSPP

f_search / problems / i_1_spp
PROBLEM SPP

Purpose

  1. Defines a one-to-one shortest path problem: single start to single goal on a grid.
  2. Composes ProblemSearch (grid + successors) with HasStart and HasGoal mixins.
  3. Constructor accepts only Grid (not Grid | str). Pickle and grid reload inherited from ProblemSearch.

h_start

@property
def h_start(self) -> int:
  1. Returns Manhattan distance between start and goal.
  2. Delegates to self.start.distance(other=self.goal).

norm_h_start

@property
def norm_h_start(self) -> float:
  1. Returns normalized distance in [0, 100].
  2. Formula: h_start / (rows + cols - 2) * 100.
  3. Returns 0.0 for trivial (1-cell) grids.

Usage Examples

Example 1 — Create via Factory ▼ 
from f_search.problems.i_1_spp import ProblemSPP

problem = ProblemSPP.Factory.without_obstacles()
print(problem.start)   # State at grid[0][0]
print(problem.goal)    # State at grid[0][3]
print(problem.grid)    # 4x4 GridMap

Example 2 — Reverse a Problem ▼ 
from f_search.problems.i_1_spp import ProblemSPP

problem = ProblemSPP.Factory.without_obstacles()
reversed_problem = problem.reverse(name='Reversed')

assert reversed_problem.start == problem.goal
assert reversed_problem.goal == problem.start
assert reversed_problem.grid is problem.grid
