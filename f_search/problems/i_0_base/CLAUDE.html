ProblemSearch

f_search/problems/i_0_base/

A. Purpose

  1. Base class for all grid-based search problems.
  2. Stores a grid reference and a pickle-safe grid name (_name_grid).
  3. Generates successor states via grid.neighbors(); supports lightweight pickling by excluding the heavy grid.

  1. Accepts a Grid (GridMap) and an optional name string.

grid (property)

@property

  1. Returns the problem's grid.
  2. Raises ValueError if grid is None (not yet restored after unpickling).

name_grid (property)

@property
def name_grid(self) -> str
  1. Returns the grid's name string.
  2. Survives pickle; stored independently from the grid object.

successors

def successors(self, state: State) -> list[State]
  1. Returns valid successor states from the given state.
  2. Delegates to grid.neighbors(cell=state.key).
  3. Wraps each neighbor cell as a StateCell.

to_analytics

def to_analytics(self) -> dict
  1. Delegates to self.grid.to_analytics() and returns the resulting dict.
  2. Subclasses extend the returned dict with additional keys.

load_grid

def load_grid(self, grids: dict[str, Grid]) -> None
  1. Restores the grid reference after unpickling.
  2. Looks up _name_grid in the provided dict of grids.

__getstate__

def __getstate__(self) -> dict
  1. Custom pickle hook; copies __dict__ with _grid set to None.
  2. Keeps _name_grid intact so the grid can be restored later.

E. Usage Examples

Successor generation 
from f_search.problems.i_0_base import ProblemSearch
from f_search.ds.state import StateBase

problem = ProblemSearch.Factory.grid_3x3()
cell_00 = problem.grid[0][0]
state_00 = StateBase(key=cell_00)
successors = problem.successors(state=state_00)
# Returns neighbors of (0,0) as StateCell objects
Pickle round-trip with grid restoration 
import pickle
from f_search.problems.i_0_base import ProblemSearch

problem = ProblemSearch.Factory.grid_3x3()
grid = problem.grid

# Pickle excludes the heavy grid
data = pickle.dumps(problem)
loaded = pickle.loads(data)
assert loaded._grid is None
assert loaded.name_grid == '3x3'

# Restore grid from a name-keyed dict
loaded.load_grid(grids={grid.name: grid})
assert loaded.grid == grid
