04

Corner-cutting benchmark-faithful: strict no-cut

Rule

Move (1,0)→(0,1). Flanks = (0,0) and (1,1). One flank blocked ⇒ illegal — the path would clip the obstacle's corner.

both flanks free → legal
either blocked → illegal

Matches MovingAI / GPPC / JPS released optimal costs.

05

√2 cost model scaled-int, not float

Recommended
CARD 10000 · DIAG 14142

octile = 10000·(dx+dy) − 5858·min(dx,dy)
same constants in cost and h ⇒ admissible & consistent. Report true dist = cost / unit.

Why not float √2

OMSPP / kA* needs exact co-optimal ties (k-shortest, INC-vs-AGG degeneracy). Float rounding makes equal-cost paths compare unequal — silently corrupting tie counts. Int keeps every compare exact and preserves the int hot-path invariant.
