sheets

f_google/services/sheets/
SUB-PACKAGE

Purpose

  1. Sub-package grouping Google Sheets service wrappers.
  2. Re-exports Spread and Sheet for convenient access.
  3. Provides spreadsheet-level and worksheet-level access to Google Sheets via gspread.

Usage Examples

Example 1: Open and Read Spreadsheet ▼ 
from f_google.services.sheets import Spread

spread = Spread.Factory.valdas(id_spread='<SPREADSHEET_ID>')
print(spread.name)        # spreadsheet title
sheet = spread['Sheet1']
print(len(sheet))         # number of rows
print(sheet[0])           # first row as list[str]
Example 2: Iterate All Sheets ▼ 
from f_google.services.sheets import Spread

spread = Spread.Factory.rami(id_spread='<SPREADSHEET_ID>')
for sheet in spread.sheets:
    print(f'{sheet.name}: {len(sheet)} rows')
Example 3: Test Spreadsheet ▼ 
from f_google.services.sheets import Spread

spread = Spread.Factory.valdas_test()
sheet = spread.sheets[0]
print(sheet[0])           # first row of first sheet
