Spread

f_google/services/sheets/spread/
CLASS

Purpose

  1. Google Spreadsheet wrapper built on gspread.
  2. Provides access to sheets within a spreadsheet by name or index.
  3. Accepts both OAuth and Service-Account credentials.
  4. Inherits HasName to expose the spreadsheet title.

Usage Examples

Example 1: Open and List Sheets ▼ 
from f_google.services.sheets.spread import Spread

spread = Spread.Factory.valdas(id_spread='<SPREADSHEET_ID>')
print(spread.name)       # spreadsheet title
print(len(spread.sheets)) # number of worksheets
Example 2: Access Sheet by Name ▼ 
from f_google.services.sheets.spread import Spread

spread = Spread.Factory.valdas_test()
sheet = spread['Sheet1']
value = sheet[1][0]  # row 1, column 0
Example 3: Service-Account Access ▼ 
from f_google.services.sheets.spread import Spread

spread = Spread.Factory.rami(id_spread='<SPREADSHEET_ID>')
for sheet in spread.sheets:
    print(sheet.name)
