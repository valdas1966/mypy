OAuth

f_google/creds/oauth/
STATIC UTILITY

Purpose

  1. Static utility class for Google OAuth (browser-based) authentication.
  2. Handles token caching, refresh, and browser login flow automatically.
  3. Factory provides pre-configured credentials for the VALDAS account.

Public API

class OAuth ▼

Class Attribute

Factory: type | None = None
  1. Factory class for creating credential instances. Attached via __init__.py.

Static Method

@staticmethod
def get_creds(path_client: str,
path_token: str,
scopes: list[str]) -> OAuthCredentials
  1. Return OAuth credentials with automatic token management.
  2. Loads cached token from path_token if it exists.
  3. Refreshes expired token using Request() if refresh token is available.
  4. Opens browser login via InstalledAppFlow if no valid token exists.
  5. Saves obtained token to path_token for future use.

Factory Method

@staticmethod
def valdas() -> OAuthCredentials
  1. Return OAuth credentials for VALDAS.
  2. macOS: reads from ~/prof/valdas/oauth_client.json and ~/prof/valdas/token.json.
  3. Other platforms: reads OAUTH_CLIENT_JSON_PATH and VALDAS_TOKEN_PATH env vars.
  4. Scopes: cloud-platform, drive, spreadsheets, presentations.

Usage Examples

Example 1: Via Factory ▼ 
from f_google.creds.oauth import OAuth

  1. From _tester.py — verifies credential is OAuthCredentials instance.
Example 2: Generic Usage ▼ 
from f_google.creds.oauth import OAuth

creds = OAuth.get_creds(
    path_client='/path/to/client_secrets.json',
    path_token='/path/to/token.json',
    scopes=['https://www.googleapis.com/auth/drive']
)
# Opens browser on first run, caches token for subsequent calls
