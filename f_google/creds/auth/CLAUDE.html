Auth

f_google/creds/auth/
STATIC UTILITY

Purpose

  1. Static utility class for Google Service-Account authentication.
  2. Takes a JSON key file path and scopes, returns SACredentials.
  3. Factory provides pre-configured credential for the RAMI service account.

Public API

class Auth ▼

Class Attribute

Factory: type | None = None
  1. Factory class for creating credential instances. Attached via __init__.py.

Static Method

@staticmethod
def get_creds(path: str, scopes: list[str]) -> SACredentials
  1. Return Service-Account credentials from a JSON key file.
  2. Delegates to SACredentials.from_service_account_file().

Factory Method

@staticmethod
def rami() -> SACredentials
  1. Return credentials for RAMI (project: noteret).
  2. macOS: reads from ~/prof/rami.json.
  3. Other platforms: reads RAMI_JSON_PATH env var.
  4. Scopes: cloud-platform, drive, spreadsheets, presentations.

Usage Examples

Example 1: Via Factory ▼ 
from f_google.creds.auth import Auth

creds = Auth.Factory.rami()
print(creds.project_id)  # 'noteret'
  1. From _tester.py — verifies project_id == 'noteret'.
Example 2: Generic Usage ▼ 
from f_google.creds.auth import Auth

creds = Auth.get_creds(
    path='/path/to/service-account-key.json',
    scopes=['https://www.googleapis.com/auth/cloud-platform']
)
