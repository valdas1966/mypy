creds

f_google/creds/
SUB-PACKAGE

Purpose

  1. Sub-package grouping all Google authentication credential providers.
  2. Re-exports Auth and OAuth for convenient access.
  3. Two strategies: Service-Account (non-interactive) and OAuth (browser-based).

Public API

Auth — Service-Account ▼
@staticmethod
def get_creds(path: str, scopes: list[str]) -> SACredentials
  1. Return SA credentials from a JSON key file.
@staticmethod
def Factory.rami() -> SACredentials
  1. Pre-configured credentials for RAMI (project: noteret).
OAuth — Browser-Based ▼
@staticmethod
def get_creds(path_client: str,
path_token: str,
scopes: list[str]) -> OAuthCredentials
  1. Return OAuth credentials with automatic token management.
  2. Loads cached token, refreshes if expired, or opens browser login.
@staticmethod
def Factory.valdas() -> OAuthCredentials
  1. Pre-configured credentials for VALDAS.

Security — Credential File Storage

  1. Credential files (JSON keys, tokens) are stored outside the repo.
  2. No credential files are tracked by git.
  3. Paths resolved via home directory or environment variables.
FilemacOS PathOther Platforms

Usage Examples

Example 1: Service-Account ▼ 
from f_google.creds import Auth

creds = Auth.Factory.rami()
print(creds.project_id)  # 'noteret'
Example 2: OAuth ▼ 
from f_google.creds import OAuth

creds = OAuth.Factory.valdas()
# Opens browser on first run, caches token for future calls
